# People Analytics — IBM HR Dataset

## Sobre o Projeto
Dashboard de análise de dados voltado para Recursos Humanos, desenvolvido com base no dataset público IBM HR Analytics. O projeto tem como objetivo identificar padrões de turnover, satisfação e perfil demográfico dos colaboradores, gerando insights estratégicos para tomada de decisão em RH.

## Ferramentas Utilizadas
- Python (Pandas, Deep Translator) — tradução e preparação dos dados
- Power BI Desktop — modelagem, DAX e visualização
- Power Query — transformação e limpeza dos dados
- GitHub — versionamento e portfólio

## Páginas do Dashboard
- **People Analytics** — visão geral com KPIs principais
- **Visão Geral** — indicadores consolidados por departamento
- **Turnover** — análise de evasão por cargo, tempo de casa e departamento
- **Satisfação** — engajamento por faixa salarial, gênero e faixa etária
- **Perfil Demográfico** — composição da força de trabalho

## Principais Insights
- Taxa de turnover de 16%, acima da média de mercado
- Representante de Vendas é o cargo com maior evasão (40%)
- Funcionários desligados ganhavam em média USD 2.050 a menos
- Quanto menor o salário, maior a taxa de turnover
- Sênior domina 52% da força de trabalho

## Dataset
IBM HR Analytics Employee Attrition & Performance — dataset público disponível no GitHub e Kaggle.
